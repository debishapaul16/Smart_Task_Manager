# Smart Task Manager

A full-stack task management application that helps users organize, track, and manage daily tasks efficiently.

Features:
- User Authentication
- Create, Update, Delete Tasks
- Task Priorities
- Due Dates
- Progress Tracking
- Responsive UI

Tech Stack:
- React.js
- Node.js
- Express.js
- MongoDB
